## 민재
```java
class Solution {
    public int[] solution(int[] arr) {
        int[] answer = new int[arr.length];
        int count = 0;
        for(int i = arr.length-1; i >= 0; i-- ){
            answer[count] = arr[i];
            count++;
        }
        return answer;
    }
}
```
